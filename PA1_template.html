Imputing missing values

Note that there are a number of days/intervals where there are missing values (coded as NA). The presence of missing days may introduce bias into some calculations or summaries of the data.

1. Calculate and report the total number of missing values in the dataset 

dim(activity_data[!complete.cases(activity_data),])
## [1] 2304    3

There are 2304 missing values in the dataset, which means that approximately 13% of the observations is incomplete.

2. Devise a strategy for filling in all of the missing values in the dataset. Create a new dataset that is equal to the original dataset but with the missing data filled in.

The missing values were imputed using the median value for each of the 5-minute intervals. For instance, if the median for the 215 interval was 0, all the NA’s for that interval were replaced by 0.

library(plyr)
imputed_data <- ddply(activity_data, .(interval), mutate, int_median = median(steps, na.rm = T))
imputed_data$steps <- ifelse(is.na(imputed_data$steps), imputed_data$int_median, imputed_data$steps)
imputed_data$int_median <- NULL

Are there differences in activity patterns between weekdays and weekends?

For this part the weekdays() function may be of some help here. Use the dataset with the filled-in missing values for this part.

1. Create a new factor variable in the dataset with levels “weekday” and “weekend” indicating whether a given date is a weekday or weekend day.

To determine whether a date is a weekday or not, we used format() funtion with the “%u” argument. Each date was assigned a value 1-7, i.e. Monday - 1, Sunday - 7.

imputed_data$wdayID <- format(imputed_data$date, "%u")
imputed_data$wday[imputed_data$wdayID %in% c("6","7")] <- "weekend"
imputed_data$wday[imputed_data$wdayID %in% c("1","2","3","4","5")] <- "weekday"
